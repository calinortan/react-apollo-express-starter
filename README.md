# React & Apollo starter kit

## Run app

In order to run the project run these scripts in this order:

### `cd server`

### `yarn`

### `yarn start:db`

### `yarn start:server`

### `cd ../client`

### `yarn`

### `yarn start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
